# UPSC Mindmap Gallery

A fully responsive, swipeable mindmap gallery built with React, Vite, Swiper, and html2canvas.

## Setup Instructions
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

## Deployment
This project is ready to be deployed on Vercel. 
Just connect your GitHub repository to Vercel, and the provided `vercel.json` will automatically handle SPA routing.
